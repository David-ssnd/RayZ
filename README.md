# RayZ

RayZ is a laser tag project, which is also a part of my school leaving exam. This repository contains the main project structure with multiple submodules for embedded devices and a Next.js website.

## Project Structure

This project consists of the following submodules:

### Web Components

- **web/frontend** - Frontend web application built with Next.js for the laser tag game interface and user management
- **web/backend** - Backend server handling game logic, player data, and API endpoints

### ESP32 Embedded Devices

- **esp32/weapon** - Firmware for the ESP32-based laser tag weapon devices
- **esp32/target** - Firmware for the ESP32-based target devices

## About

This laser tag system combines modern web technologies with embedded hardware to create an interactive gaming experience. The ESP32 microcontrollers power the physical laser tag equipment, while the web application provides game management, scoring, and player interaction.
